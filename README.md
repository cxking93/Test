# vue3.0 QuickStart  开箱即用的3.0模板


## 使用方法

```
npm i 
```

>注：vue-cli 3.0版本不要使用cnpm  会造成无法热更新

```
npm run serve
```

## 打包

```
npm run build
```

- [x] css预处理器less
- [x] 封装axios
- [x] vue.config.js配置
- [x] vuex
- [x] router
- [x] normalize.css
